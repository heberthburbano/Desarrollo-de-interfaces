# Finanzas Personales SPA (Fase 1)

Esta es una Single Page Application (SPA) de finanzas personales construida con **HTML5 Semántico, CSS3 y JavaScript Vanilla**.

## Estructura del Proyecto

- `index.html`: Punto de entrada único. Contiene todas las secciones (`#home`, `#transactions`, etc.) ocultas/visibles según la navegación.
- `style.css`: Estilos modernos con enfoque **Mobile First**, Grid/Flexbox y Variables CSS.
- `router.js`: Clase encarga de gestionar la navegación y el historial del navegador.
- `main.js`: Lógica de inicialización y eventos de la interfaz.

## Cómo Ejecutar

Al ser un proyecto estático sin dependencias de Node.js ni bundlers, puedes abrirlo directamente:

1. Navega a la carpeta del proyecto.
2. Haz doble clic en el archivo `index.html` para abrirlo en tu navegador favorito.
3. O usa una extensión como **Live Server** en VS Code para recarga automática.

## Características (Fase 1)

- **Navegación SPA**: Sin recargas de página.
- **Diseño Responsivo**:
  - Móvil: Navegación inferior fija + FAB.
  - Escritorio: Barra lateral y layout Grid.
- **Interfaz Moderna**: Tarjetas con sombras, tipografía limpia (Outfit) y modo visual agradable.

## Tecnologías
